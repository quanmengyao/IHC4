# IHC4
# Radiogenomic Model for Breast Cancer Recurrence Prediction

This repository contains the code used for the radiogenomic analysis described in the manuscript:

**"Radiogenomic model for predicting recurrence risk in breast cancer."**

The study integrates radiomic features extracted from imaging with molecular information (IHC4 score) to develop a radiogenomic signature for recurrence risk prediction.

## Repository Structure
code/
spearman_analysis.R
feature_selection.R
main_svm.py
nomogram_cox.R
data/
Results_Features.xlsx
## Data
The shared dataset contains anonymized radiomic features and clinical variables used for model development.
All patient identifiers were removed prior to data sharing.
